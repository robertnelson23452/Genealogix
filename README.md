# 🌳 Genealogix - Decentralized Ancestry Tree

> NFT-based family lineage mapping on the Stacks blockchain

## 📖 Overview

Genealogix is a revolutionary smart contract that enables users to create, manage, and verify family trees as NFTs on the Stacks blockchain. Each family member becomes a unique NFT with verifiable relationships, creating an immutable and decentralized ancestry record.

## ✨ Features

- 🎯 **NFT Family Members**: Each person in the family tree is represented as a unique NFT
- 👨‍👩‍👧‍👦 **Family Relationships**: Track parents, children, and spouses with verifiable connections
- 🔍 **Lineage Verification**: Community-driven verification system for authenticity
- 📊 **Generation Tracking**: Automatic generation counting and lineage path mapping
- 🏆 **Ownership Control**: Only NFT owners can modify their family member's relationships
- 📈 **Multi-generational Support**: Build comprehensive family trees across generations
